**Автор** Любовь Сухочева
**Дата сдачи:** 15.05.2026

## Описание программы
Desktop-приложение на Python с графическим интерфейсом (GUI) для ведения каталога личных фильмов. Программа позволяет сохранять названия, жанры, годы выпуска и оценки фильмов. Данные автоматически сохраняются в локальный файл формата JSON.

## Функционал
- Добавление фильмов через форму с валидацией ввода.
- Защита от некорректных данных: проверка числового значения года и диапазона рейтинга (0–10).
- Просмотр списка фильмов в интерактивной таблице.
- Фильтрация базы данных по жанру (поиск по подстроке) и году выпуска.
- Автоматическое сохранение/загрузка базы из файла `movies.json`.

## Требования для запуска
Что нужно установить на компьютер:
- Python 3.6 или выше
- Установленная библиотека Tkinter

## Инструкция по запуску
1. Откройте терминал или командную строку
3. Запустите приложение:
   ```bash
   python main.py
   ```

## Примеры использования

### Добавление фильма
1. Введите в поля:
    **Название:**  `Начало`
    **Жанр:** `Фантастика`
    **Год:** `2010`
    **Рейтинг:** `8.8`.
2. Нажмите **Добавить фильм**.
3. Фильм появится в списке

### Фильтрация
1. Добавьте два фильма с жанрами `Комедия` и `Боевик`.
2. В блоке фильтрации в поле "По жанру" введите `ком`.
3. Нажмите **Применить**.
4. В таблице останется только фильм с жанром `Комедия`.
